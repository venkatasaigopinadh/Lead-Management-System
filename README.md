📌 Lead Management System

A beginner-friendly full stack web application built using Flask and SQLite to manage customer leads efficiently.

🚀 Features
Lead Submission Form
Store Leads in Database
Dashboard to View Leads
Search & Filter Leads
Update Lead Status
Responsive Clean UI
Form Validation

🛠️ Tech Stack
Python
Flask
SQLite
HTML/CSS
Bootstrap

📂 Modules
Lead Form Page
Database Integration
Admin Dashboard
Search Functionality
Status Management

▶️ How to Run
python app.py

Open browser:
http://127.0.0.1:5000
http://127.0.0.1:5000/dashboard
